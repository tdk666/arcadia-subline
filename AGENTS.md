# AGENTS.md — Système multi-agent

Ce repo est conçu pour la collaboration entre agents IA (Claude Code, Copilot, Codex).

## Lire d'abord

**→ Consulter [`CLAUDE.md`](CLAUDE.md)**

Le fichier `CLAUDE.md` contient :
- la séparation code / mémoire
- les points d'entrée (source-registry, decision-log, session-log)
- les conventions de travail
- les principes MECE

## Flux standard

```
1. CLAUDE.md (orientation)
   ↓
2. docs/studio/README.md (vue globale)
   ↓
3. session-log.md (où on en était)
   ↓
4. decision-log.md + source-registry.md (contexte validé)
   ↓
5. Travailler + mettre à jour
```

## Pas de setup nécessaire

- Aucune dépendance ajoutée
- Zéro configuration
- Utiliser les templates Obsidian si besoin (`.obsidian/templates/`)
