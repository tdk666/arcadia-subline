# Arcadia Subline — Studio

Espace de travail léger pour les synthèses, décisions et historiques de sessions réservés aux agents IA (Codex / Claude).

But de ce dossier

- centraliser les synthèses courtes et les décisions d'architecture
- fournir une mémoire de travail réutilisable par les agents sans recharger tout le repo
- stocker les enregistrements de sessions et la registry des sources

Structure recommandée

- docs/studio/ : exports et synthèses versionnées avec le code
- .obsidian/templates/ : templates Obsidian pour sessions, features, décisions
- notes/ : notes de travail rapides (optionnel, pour usage local dans le vault)
